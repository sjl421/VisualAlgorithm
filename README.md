## Dependencies

- Python 3
- PyQt5
- QScintilla2

For Ubuntu users:
```sh
sudo apt-get install python3 python3-pyqt5 python3-pyqt5.qsci
```

For Arch Linux users:
```sh
pacman -S python-pyqt5 python-qscintilla-qt5
```

## Clone & Run

```sh
git clone https://github.com/zlsun/VisualAlgorithm.git
cd VisualAlgorithm
make
```
